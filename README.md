# C# Fundamentals Projects

This repository contains a collection of small C# console applications developed to demonstrate core programming concepts.

##  Overview

The projects in this repository focus on fundamental concepts in C# such as:

* Selection statements (if, else-if, switch)
* Iteration (while, do-while, for loops)
* Input validation
* Methods and class structure
* Basic algorithms

##  Projects Included

### 1. Cost Calculator

Calculates the total cost of a purchase based on quantity and applies discounts using conditional logic.

### 2. Guess the Number Game

A simple number guessing game with multiple difficulty levels (Easy, Medium, Hard) using random number generation and loops.

### 3. Temperature Converter

Converts temperatures between Celsius and Fahrenheit and displays results in a structured table format using a menu-driven interface.

### 4. String Functions

Performs basic string operations such as:

* Calculating string length
* Converting text to uppercase
* Displaying messages based on day input using switch statements

### 5. Scheduler

Generates work schedules based on intervals:

* Weekend shifts (every 2 weeks starting from week 2)
* Night shifts (every 4 weeks starting from week 1)

1. Clone the repository:

   ```
   git clone https://github.com/jasminerezai/csharp-fundamentals-projects.git
   ```

   ```
   Assignment2_MAU.sln
   ```

3. Select a project and run it.
